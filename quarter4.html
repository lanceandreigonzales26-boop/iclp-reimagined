Topic 1

Oxidation States

Oxidation states (or oxidation numbers) are assigned charges that track electron transfer in a reaction.

01 / RULES

Oxidation State Rules

1. Pure element = 0 (e.g., O₂, Fe, Na)
2. Monatomic ion = its charge (Na⁺ = +1, Cl⁻ = −1)
3. Oxygen = −2 (except in peroxides: −1, and OF₂: +2)
4. Hydrogen = +1 (−1 in metal hydrides)
5. Sum of oxidation states in a neutral compound = 0
6. Sum of oxidation states in a polyatomic ion = ion charge

02 / WORKED EXAMPLE

Finding Oxidation States

Find the oxidation state of S in H₂SO₄:

H is +1, O is −2. Let S = x.
2(+1) + x + 4(−2) = 0
2 + x − 8 = 0
x = +6

So sulfur has an oxidation state of +6 in sulfuric acid.

03 / COMMON VALUES

Common Oxidation States

Fluorine: always −1
Oxygen: usually −2
Hydrogen: usually +1
Group 1 metals: always +1
Group 2 metals: always +2
Transition metals: variable (e.g., Fe can be +2 or +3; Cr can be +2, +3, or +6)

Topic 2

Identifying Redox Reactions

A reaction is a redox reaction if and only if oxidation states change between reactants and products.

01 / OIL RIG

OIL RIG Mnemonic

Oxidation Is Loss (of electrons)
Reduction Is Gain (of electrons)

When an atom loses electrons, its oxidation state increases (it is oxidized).
When an atom gains electrons, its oxidation state decreases (it is reduced).

Oxidation and reduction always occur together — you cannot have one without the other.

02 / AGENTS

Oxidizing & Reducing Agents

Oxidizing agent: Causes another species to be oxidized. It itself is reduced (gains electrons). It is the electron acceptor.

Reducing agent: Causes another species to be reduced. It itself is oxidized (loses electrons). It is the electron donor.

Memory trick: The oxidizing agent gets reduced; the reducing agent gets oxidized.

03 / EXAMPLE

Identifying in a Reaction

Zn + CuSO₄ → ZnSO₄ + Cu

Zn: 0 → +2 (oxidation state increases) → oxidized, acts as reducing agent

Cu: +2 → 0 (oxidation state decreases) → reduced, CuSO₄ acts as oxidizing agent

Electrons transferred from Zn to Cu²⁺.

Topic 3

Half-Reaction Method

A systematic approach to balancing redox equations by separating oxidation and reduction into two independent half-reactions.

01 / STEPS

Steps for Half-Reaction Balancing

1. Assign oxidation states to identify what's oxidized and reduced.
2. Write two half-reactions (oxidation and reduction).
3. Balance atoms other than O and H first.
4. Balance O by adding H₂O; balance H by adding H⁺.
5. Balance charge by adding electrons (e⁻).
6. Multiply half-reactions so electrons cancel.
7. Add half-reactions and simplify.

02 / EXAMPLE

Worked Half-Reaction Example

Balance: MnO₄⁻ + Fe²⁺ → Mn²⁺ + Fe³⁺ (acidic)

Reduction: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O
Oxidation: Fe²⁺ → Fe³⁺ + e⁻

Multiply oxidation by 5: 5Fe²⁺ → 5Fe³⁺ + 5e⁻

Final: MnO₄⁻ + 5Fe²⁺ + 8H⁺ → Mn²⁺ + 5Fe³⁺ + 4H₂O

Term Definition Oxidation State Change
Oxidation Loss of electrons Increases (more positive)
Reduction Gain of electrons Decreases (more negative)
Oxidizing Agent Accepts electrons; causes oxidation of another Its own state decreases (it is reduced)
Reducing Agent Donates electrons; causes reduction of another Its own state increases (it is oxidized)
